# 使用手册 (User Manual)

## 界面概览

The Principia 的界面主要分为三个区域：
1.  **左侧面板 (编辑器)**: 输入 LaTeX 或 Markdown 笔记的地方。
2.  **中间面板 (画布)**: 用于手写和绘图的数字白板。
3.  **右侧面板 (预览)**: 笔记的实时格式化视图。

## 1. 编辑与格式化

### 切换模式
-   点击顶部导航栏的 **TEX / MD** 切换按钮。
-   **注意**: 切换模式会触发 AI 转换，AI 将重写您的内容以适应新格式，同时保留标题、列表结构和**文本颜色**。

### 编写内容
-   **LaTeX 模式**: 使用标准 LaTeX 语法 (例如 `\section{}`, `\textbf{}`, `\textcolor{red}{}`)。
-   **Markdown 模式**: 使用标准 Markdown (例如 `#`, `**`, `<font color="red">`)。
-   **数学公式**: 在两种模式下，都使用 `$...$` 表示行内公式，`$$...$$` 表示块级公式。

## 2. 手写与 OCR 识别

### 工具栏
-   **画笔 (Pen)**: 绘制笔迹。
-   **橡皮擦 (Eraser)**: 擦除笔迹。
-   **调色板**: 更改画笔颜色。AI 在识别时会检测此颜色。
-   **背景**: 切换空白、横线或网格背景。
-   **页面**: 使用 `+` 按钮添加新画布页。

### 识别流程
1.  在画布上书写公式或文本。
2.  点击画布工具栏中的 **闪烁图标 (Recognize)**。
3.  AI 将把您的手写内容转录为 LaTeX 并插入到编辑器光标位置。
    -   *提示*: 红色的笔迹会被转录为 `\textcolor{red}{...}`。

## 3. AI 分析与可视化

1.  **选择文本**: 在编辑器或预览区高亮选中任何文本或公式。
2.  **点击 "Analyze"**: 会出现一个悬浮徽章，点击它。
3.  **查看结果**:
    -   **解释 (Explanation)**: 针对选中概念的简明物理/数学解释。
    -   **可视化 (Visualization)**: 如果适用，下方会加载一个交互式模拟。通常包含可调节参数的滑块。

## 4. 导出 (Export)

点击头部的 **Export** 按钮打开导出对话框。
-   **格式**: 选择将主文档导出为 `.tex` 或 `.md`。
-   **包含 PDF**: 勾选此项将把您的手写画布编译为 PDF 文件。
-   **包含分析**: 勾选此项将包含 AI 生成的解释和交互式 HTML 可视化文件。
-   **下载**: 生成一个包含所有资源的 `.zip` 压缩包。

## 5. 设置 (Settings)

点击头部的 **齿轮图标** 配置 AI 提供商。
-   **推理模型 (Reasoning Model)**: 用于解释和格式转换。
-   **视觉模型 (Vision Model)**: 用于手写 OCR 识别、可视化生成。
-   **配置**: 输入您的 `Base URL` 和 `API Key`。兼容 OpenAI、Gemini 以及其他 OpenAI 兼容接口。
-   **语言设置**: 点击语言图标切换中英文界面。

## 6. 图片公式识别

### 图片上传
1.  点击画布工具栏中的 **图片图标 (Upload Image)**。
2.  选择包含数学公式的图片文件。
3.  系统会自动处理图片并识别其中的公式。

### 识别流程
1.  上传图片后，系统会使用 AI 识别图片中的数学公式。
2.  识别结果会以 LaTeX 格式插入到编辑器中。
3.  您可以直接编辑生成的 LaTeX 代码。

### 注意事项
- 建议上传清晰、光线充足的公式图片
- 复杂的公式可能需要多次尝试以获得最佳识别效果

## 7. 函数图像绘画功能增强

### 多函数绘制
- 支持在同一坐标系中绘制多个函数
- 每个函数会使用不同颜色区分

### 参数调整
- 提供直观的滑块控件调整函数参数
- 实时预览参数变化对图像的影响

### 实时数据显示
- 显示函数的关键数据点
- 提供函数的导数、积分等相关信息

### 缩放平移
- 使用鼠标滚轮进行缩放
- 拖动画布进行平移
- 双击画布重置视图

## 8. 历史记录功能

### 访问历史记录
1.  点击头部的 **时钟图标 (History)**。
2.  在历史记录面板中查看所有保存的内容。

### 历史记录管理
- **搜索**: 使用搜索框快速查找历史记录
- **加载**: 点击历史记录项右侧的加载图标，将内容加载到编辑器
- **删除**: 点击历史记录项右侧的删除图标，删除指定记录
- **清除**: 点击面板顶部的清除按钮，删除所有历史记录

### 自动保存
- 导出内容时会自动保存到历史记录
- 最多保存最近 50 条历史记录

## 9. 常用数学公式表格

### 访问公式表格
1.  点击头部的 **书籍图标 (Formulas)**。
2.  在公式表格面板中浏览不同类别的数学公式。

### 公式分类
- **代数**: 二次方程、平方和、平方差、二项式定理等
- **微积分**: 导数、积分、链式法则、乘积法则等
- **三角学**: 勾股定理、正弦加法、余弦加法、正切公式等
- **物理学**: 牛顿第二定律、动能、势能、欧姆定律等

### 一键插入
1.  在公式表格中选择需要的公式
2.  点击公式右侧的插入图标
3.  公式会以 LaTeX 格式插入到编辑器中

## 10. 性能优化

### 缓存机制
- 系统会缓存已生成的物理动态模型
- 重复请求相同内容时会直接使用缓存结果
- 提高模型生成速度，减少等待时间

### 渲染优化
- 前端使用 React 性能优化技术
- 减少不必要的重新渲染
- 提高复杂公式和图像的渲染速度

## 11. 本地化支持

### 语言切换
- 点击设置面板中的语言图标
- 支持中英文界面切换
- 所有生成内容会根据选择的语言自动调整

### 中文支持
- 界面元素完整汉化
- 生成的解释和可视化文本为中文
- 符合中文表达习惯的专业术语
